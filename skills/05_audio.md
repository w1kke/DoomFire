# Skill: Audio (crackle loop)

## Goal
Play a looped crackle track with WebAudio, respecting browser autoplay rules.

## Tasks
- Load the mp3/ogg asset.
- Only start audio after a user gesture ("Enable sound" or first Ignite).
- Loop playback with short fades to avoid clicks.
- Modulate volume with intensity/heat.
